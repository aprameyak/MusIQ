## About

**MusicRatingApp** is a native iOS app for rating and discovering music. Users rate albums, songs, and artists (1-10), see global rankings, build taste profiles, and connect with friends.

---

## Features

- User authentication (Email, Apple, Google, Spotify)  
- Rate albums, songs, and artists  
- Global rankings and trend tracking  
- Personalized taste profiles with charts  
- Social features: friends, compatibility scores, sharing  
- Real-time notifications  

---

## Technology Stack

- **Frontend**: SwiftUI, Combine, Swift 5+, Keychain Services  
- **Backend**: Node.js, Express.js, TypeScript, PostgreSQL, Redis
