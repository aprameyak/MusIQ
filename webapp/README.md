# MusIQ Web App

Minimal web app for iOS App Store support and privacy page requirements.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Landing page
- `/support` - Support page
- `/privacy` - Privacy policy page

## Build

```bash
npm run build
npm start
```
